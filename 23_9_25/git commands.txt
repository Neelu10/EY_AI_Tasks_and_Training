BASIC GIT COMMANDS

Setup-

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"


Repository management-

git init
git clone <repo url>


Working with files-

git add .
git status

Commiting changes-

git commit -m "first commit"
git commit -am "message"

Merge & Branch-

git branch
git branch <name>
git merge <name>
git checkout <name>

Remote Repositories-

git remote -v                
git remote add origin <url> 
git push origin <branch>   
git pull origin <branch>    


Log files-

git log                     
git log --oneline            
git diff                    
git diff --staged

Fetch commands-

git fetch 
git fetch origin
git fetch --all
git fetch origin branch_name



Undo changes-

git checkout -- <file>       
git reset HEAD <file>
